## Notion References

- The page formerly known as "Learning HQ" has been renamed to **AI Learning**. Always reference and search for it as "AI Learning".

## Session Summary Notes

At the end of every session, create a summary note in the **AI Learning** database in Notion.

### Note Title Format

`AI Learning DD-MM-YYYY`

Example: `AI Learning 02-07-2026`

If a note for today's date already exists, append to it instead of creating a new one.

### Content Guidelines

- Summarize key topics discussed, decisions made, and actions taken during the session.
- Include the **Answer Reference Prefix** for each item summarized, so it's clear which conversation or reference the summary came from and can be tracked easily.
- Write concisely but with enough detail to be useful when reviewed later.
